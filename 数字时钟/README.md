# 数字时钟

## 思路
在定时器中不断的获取当前时间并转成时分秒格式</br>
使用transform属性改变每一列纵向位置，让当前时间对应的数字居中显示</br>
通过当前时间的数字与每一个数字的差来为不同的数字添加不同的CSS样式，让越远的数字的透明度越大
